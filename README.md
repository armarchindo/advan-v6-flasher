# advan-v6-flasher
Flasher firmware MOD untuk modem CPE Advan V6 atau ZLT S50

Cara install urut:
1. Download aplikasi bernama Termux di playstore
2. Koneksikan HP Android (wajib) ke wifi Advan V6
3. Copas command ini ke termux
```
wget https://raw.githubusercontent.com/armarchindo/advan-v6-flasher/main/advanv6_flasher -O advanv6_flasher && chmod +x ./advanv6_flasher && ./advanv6_flasher
```
4. Next (ENTER) sampai selesai dan tunggu modem restart
